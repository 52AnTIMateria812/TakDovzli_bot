# ТАК ДОВЕЗЛИ - Магазин в Telegram

**Универсальный шаблон для создания интернет-магазина внутри Telegram (Mini App).**

> Полностью готовый интерфейс и бэкенд для быстрой продажи товаров через удобное веб-приложение прямо в мессенджере.

Этот проект предоставляет современный и интуитивно понятный способ продавать товары, используя возможности **Telegram Web Apps**. 

Для покупателя работа магазина выглядит абсолютно естественно — как обычное приложение внутри мессенджера, без необходимости переходить в браузер. 

---

* **Быстрый доступ:** Покупателям **не нужно скачивать** отдельные приложения или переходить в браузер.
* **Современный UI:** Красивый каталог, корзина покупок, удобное оформление заказа и просмотр деталей товара.
* **Интерактивность:** Мгновенное добавление товаров в корзину, подсчет итоговой суммы и валидация данных покупателя.
* **Уведомления:** Бот моментально отправляет администратору (и покупателю) красивый чек заказа со всеми деталями прямо в чат.

## Пошаговая инструкция по настройке и запуску

### Шаг 1. Создание Телеграм-бота и получение токена
1. В Telegram найдите официального бота `@BotFather` и отправьте команду `/newbot`.
2. Задайте имя боту и его юзернейм (например, `tak_dovezli_bot`).
3. Скопируйте полученный **токен бота** (например, `123456789:ABCde...`).

### Шаг 2. Настройка файла `.env`
1. В корневой папке проекта создайте файл **`.env`**.
2. Откройте `.env` в Блокноте и вставьте туда данные:
```env
BOT_TOKEN=токен_вашего_бота_из_шага_1
```

---

### Шаг 3. Установка библиотек

Для работы бэкенда потребуется установить несколько Python-библиотек.

1. Откройте терминал в папке проекта.
2. **Установите зависимости:**
   Копируем всю команду целиком и запускаем:
   ```cmd
   pip install pyTelegramBotAPI python-dotenv fastapi uvicorn
   ```

---

### Шаг 4. Настройка ngrok (для локальной разработки)

Чтобы Telegram мог обращаться к вашему локальному серверу, нужен безопасный HTTPS-туннель.

1. Запустите в терминале команду:
   ```cmd
   ngrok http 8000
   ```
2. **Скопируйте полученный HTTPS-адрес.**
3. Вставьте его в файл `backend/bot.py` в переменную `WEBAPP_URL`:
   ```python
   WEBAPP_URL = "https://ваш-адрес.ngrok-free.app"
   ```

---

### Шаг 5. Запуск сервера и бота

Вам понадобятся два терминала:

1. В первом терминале запустите сервер с API и фронтендом:
   ```cmd
   python backend/server.py
   ```
2. Во втором терминале запустите самого бота:
   ```cmd
   python backend/bot.py
   ```
3. Теперь можно перейти в бота и нажать **"Открыть магазин"**!

## Подробная документация о компонентах

Так же подготовлены отдельные детальные руководства по устройству каждого модуля системы. Вы можете изучить их для глубокого понимания работы магазина:

1. [Архитектура сервера (backend/server.py)](backend/server.py) — FastAPI сервер, который отвечает за выдачу товаров (REST API) и отдачу статики (HTML/CSS/JS).
2. [Telegram Бот (backend/bot.py)](backend/bot.py) — Telegram-бот на `pyTelegramBotAPI`, который отвечает за кнопку меню и обработку веб-данных (заказов).
3. [База данных (backend/database.py)](backend/database.py) — Модуль SQLite для хранения и генерации тестовых товаров и категорий.
4. [Фронтенд (frontend/)](frontend/) — Полностью самостоятельное SPA на чистом JS/HTML/CSS для работы внутри окна Telegram Web App.
